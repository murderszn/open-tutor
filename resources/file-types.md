# 📁 The Ultimate Guide to File Types

When you work on a computer, save a document, download an image, or write code, you are working with **files**. Every file has an **extension** (the letters after the dot in the file name, like `.jpg` or `.txt`) that tells the computer what kind of data is inside it and what program should open it.

As you explore computer science, programming, and general productivity, you will encounter many different file types. Here is a master cheat sheet!

## 🗂️ Vocabulary (kid‑friendly)
- Extension: the letters after a dot in a file name that hint how to open it.
- Format: the rules for how data is stored inside a file.
- MIME type: a standard label (like image/png) telling apps what a file is.
- Encoding: how text is stored as bytes (UTF‑8 is common and safe).
- Compression: shrinking file size; lossy (smaller but some quality lost) vs lossless (no quality lost).
- Container vs codec: container holds streams (video, audio, captions). Codec is the method to encode/decode a stream.

## 📝 Word Processing & Documents
These files hold text, formatting, and sometimes images.

*   **`.pdf` (Portable Document Format)**: Created by Adobe. PDFs are designed to look exactly the same no matter what device or software you open them on. They are like a digital "final print" of a document. You usually cannot easily edit them.
*   **`.docx` / `.doc` (Microsoft Word Document)**: The standard format for Microsoft Word. If you download a Google Doc, it will often download as a `.docx` file. It supports rich text, images, and complex formatting.
*   **`.txt` (Plain Text)**: The most basic file type. It holds only letters and numbers with absolutely zero formatting (no bold, no colors, no different fonts). You can open it on literally any computer instantly.
*   **`.md` (Markdown)**: A text file (what you are reading right now!) that uses simple symbols (like `#` or `*`) to format text on platforms like GitHub or Discord. 

## 📊 Spreadsheets & Data
These files hold rows and columns of data, numbers, and lists.

*   **`.xlsx` / `.xls` (Microsoft Excel Spreadsheet)**: The standard format for Microsoft Excel. If you download a Google Sheet, it will often download as an `.xlsx` file. It supports multiple sheets, complex formulas, macros, and graphs.
*   **`.csv` (Comma Separated Values)**: A very simple, "plain text" version of a spreadsheet. Each line is a row, and commas separate the columns. This is the most common format used in data science and programming to move data between different systems because it is so lightweight.
*   **`.json` (JavaScript Object Notation)**: A file type used mostly by programmers to store and send data across the internet. It looks like a structured list with keys and values. *You will use this in your Perfect 3-Car Collection assignment!*

## 🎬 Presentations
Files used for slideshows and pitch decks.

*   **`.pptx` / `.ppt` (Microsoft PowerPoint Presentation)**: The standard format for Microsoft PowerPoint. If you download a Google Slides presentation, it will download as a `.pptx` file. 

## 💻 Code & Web Development
These files power the internet and your computer programs.

*   **`.html` (HyperText Markup Language)**: The skeleton of any website. It tells the browser what text, images, and links are on the page.
*   **`.css` (Cascading Style Sheets)**: The "paint job" for a website. It tells the browser how the `.html` should look (colors, fonts, spacing).
*   **`.js` (JavaScript)**: The "muscle" of a website. It makes websites interactive (like clicking a button, fetching data, or running games in the browser).
*   **`.py` (Python Script)**: A file containing code written in the Python programming language. You will use these often in STEM to write scripts, crunch data, or build apps.

## 🖼️ Images
*   **`.jpg` or `.jpeg` (Joint Photographic Experts Group)**: The most common format for photos. It uses "lossy" compression, meaning it sacrifices a tiny bit of quality to make the file size much smaller.
*   **`.png` (Portable Network Graphics)**: Uses "lossless" compression (higher quality than JPG) and supports **transparency**. If you have a logo with no background, it is a PNG.
*   **`.gif` (Graphics Interchange Format)**: Supports simple animations (like memes). It has low color quality but is widely used for short, looping clips.
*   **`.svg` (Scalable Vector Graphics)**: Instead of a normal grid of pixels, an SVG uses math to draw shapes. This means you can zoom in on an SVG infinitely and it will *never* get blurry or pixelated! Great for logos and icons.
