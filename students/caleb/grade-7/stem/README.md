# 🔬 7th Grade STEM

Welcome to your STEM (Science, Technology, Engineering, and Mathematics) folder! Save your experiments, coding projects, and notes here.

## Curriculum Goals (End of Year Expectations)

### 1. Physical Science (Grade 9 Track)
Accelerated survey to ensure high‑school readiness. Follow the Physical Science Roadmap and complete one module per week with a lab.

- Measurement & Units (SI, dimensional analysis) → use `resources/weights_and_measures.md`
- Matter & Atomic Structure (atoms, isotopes, periodic table)
- Bonding & Reactions (intro; simple balancing)
- Energy & Heat (temperature vs heat, specific heat)
- Motion & Forces (graphs, Newton’s Laws, friction)
- Waves & Light (properties, EM spectrum)
- Biology & Anatomy (cells; systems overview; homeostasis)
- Lab Skills & Communication (variables, error, graphing)

Roadmap & Labs:
- Assignment: `assignments/science/physical-science-roadmap/README.md`
- Lab Menu: `assignments/science/physical-science-roadmap/caleb-lab-menu.md`
- Template: `assignments/science/physical-science-roadmap/lab-report-template.md`

### 2. Computer Science & Coding
*   **Tools of the Trade**: Learn to use and navigate:
    *   Visual Studio Code (VS Code)
    *   GitHub (Version control and collaboration)
*   **Programming**: Write foundational scripts and applications using **Python**.
*   **File Types**: Read the **[Ultimate Guide to File Types](../../../../resources/file-types.md)** to understand basic computer file types used for productivity and coding (e.g., `.txt`, `.csv`, `.html`, `.py`).
*   **Web Hosting**: Start designing, building, and hosting your very own websites!

## Weekly Science Rhythm (repeat)
- Mon: Module reading + vocab
- Tue: Plan lab (materials, variables, safety)
- Wed: Run lab, collect data
- Thu: Analyze + draft report
- Fri: Finalize report + reflection

Quizzes: See weekly cumulative quizzes in `students/caleb/grade-7/stem/quizzes` (Weeks 01–12).
