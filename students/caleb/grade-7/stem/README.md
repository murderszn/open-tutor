# 🔬 7th Grade STEM

Welcome to your STEM (Science, Technology, Engineering, and Mathematics) folder! Save your experiments, coding projects, and notes here.

## Curriculum Goals (End of Year Expectations)

### 1. Physical Science
*   **Building Blocks of Matter**: Describe what an atom, a molecule, and an element are.
*   **States of Matter**: Understand the different states of matter (solid, liquid, gas, plasma) and how they transition.
*   **Properties of Matter**: Understand the difference between **mass** and **density**.

### 2. Computer Science & Coding
*   **Tools of the Trade**: Learn to use and navigate:
    *   Visual Studio Code (VS Code)
    *   GitHub (Version control and collaboration)
*   **Programming**: Write foundational scripts and applications using **Python**.
*   **File Types**: Understand basic computer file types used for productivity and coding (e.g., `.txt`, `.csv`, `.html`, `.py`).
*   **Web Hosting**: Start designing, building, and hosting your very own websites!
