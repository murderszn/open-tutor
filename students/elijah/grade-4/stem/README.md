# 🔬 4th Grade STEM

Welcome to your STEM (Science, Technology, Engineering, and Mathematics) folder! Save your experiments, coding projects, and notes here.

## Curriculum Goals (End of Year Expectations)

### 1. Physical Science
*   **Building Blocks of Matter**: Describe what an atom, a molecule, and an element are.
*   **States of Matter**: Understand the different states of matter (solid, liquid, gas) and how they change (e.g., melting, freezing).
*   **Properties of Matter**: Understand the difference between **mass** (how much stuff is in an object) and **density** (how tightly packed the stuff is).

### 2. Computer Science & Coding
*   **Tools of the Trade**: Learn to use and navigate:
    *   Visual Studio Code (VS Code)
    *   GitHub (Saving and sharing code)
*   **Programming**: Start writing basic scripts using **Python**.
*   **File Types**: Learn about basic computer file types used for work and coding (e.g., `.txt`, `.csv`, `.html`, `.py`).
*   **Web Hosting**: Start designing, building, and hosting your very own simple websites!
