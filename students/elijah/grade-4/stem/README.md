# 🔬 4th Grade STEM

Welcome to your STEM (Science, Technology, Engineering, and Mathematics) folder! Save your experiments, coding projects, and notes here.

## Curriculum Goals (End of Year Expectations)

### 1. Physical Science (Grade 6 Track)
Catch‑up survey aligned to a middle‑school level. Use the Physical Science Roadmap (Elijah track). Do simple notes + one lab per module.

- Measurement & Units (metric basics; simple conversions) → use `resources/weights_and_measures.md`
- Matter & Changes (states; melting/freezing; mixtures)
- Energy & Heat (potential vs kinetic; heat transfer)
- Motion & Forces (speed; friction; simple machines)
- Waves & Light (sound/vibration; light/reflect)
- Biology & Anatomy (cells; bones/muscles; heart/lungs)

Roadmap & Labs:
- Assignment: `assignments/science/physical-science-roadmap/README.md`
- Lab Menu: `assignments/science/physical-science-roadmap/elijah-lab-menu.md`
- Template: `assignments/science/physical-science-roadmap/lab-report-template.md`

### 2. Computer Science & Coding
*   **Tools of the Trade**: Learn to use and navigate:
    *   Visual Studio Code (VS Code)
    *   GitHub (Saving and sharing code)
*   **Programming**: Start writing basic scripts using **Python**.
*   **File Types**: Read the **[Ultimate Guide to File Types](../../../../resources/file-types.md)** to learn about basic computer file types used for work and coding (e.g., `.txt`, `.csv`, `.html`, `.py`).
*   **Web Hosting**: Start designing, building, and hosting your very own simple websites!

- Mon: Read + vocab words

- Thu: Write what happened in 3–4 sentences
- Fri: Show parents and file your page
