# Performance Matrix — As of 2026-06-05

This matrix blends completion (how much), accuracy (how well), and an interest/engagement index (where they’re leaning) using assignment volume and recency.

Method summary
- Completed = count of `students/<name>/grade-#/<subject>/assignments/*.md` with student responses.
- Accuracy = averages from graded items (see Report Cards).
- Engagement Score = volume × recency weight (recent ×2) × (1 + streak_bonus up to +10%).
- Interest Index [%] = 100 × (Engagement by subject) / (Engagement across subjects for that student).
