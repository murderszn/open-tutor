## Interactive Version

- Open the interactive mind map: `teachers/sites/mind-map/index.html`
  - Pan/zoom with your mouse or trackpad
  - Click nodes to expand/collapse branches
  - Use the top bar to Expand, Collapse, or Center the view

Tip: To preview locally, run `python3 -m http.server 8000` at the repo root, then visit:
- http://localhost:8000/teachers/sites/mind-map/index.html
