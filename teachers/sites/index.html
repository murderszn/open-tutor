Our Homeschool Operating System

The [JFD] Learning Center uniquely relies on real-world tech stacks to foster profound independence and digital literacy:

  • GitHub (The Classroom): Our curriculum, assignments, and student hubs live here as Markdown files. Students complete their work and submit "Pull Requests" for Mom and Dad to review. This acts as our permanent, immutable portfolio and gradebook.
  • Discord (The Help Desk): Real-time, async communication. We use dedicated channels (e.g., `#math-help`, `#stem-projects`) integrated with custom AI "Teacher Bots". When students are stuck, they ping the bots for Socratic guidanceвҖ”getting hints and analogies, never direct answers.

Program Requirements

Set up these accounts and tools before the first sprint. See Resources for quick links.

  • Accounts: Gmail (email), GitHub, Discord.
  • Software: VS Code editor (with GitHub signвҖ‘in).
  • Hardware: Webcam and microphone (for weekly video reviews and checkвҖ‘ins).
  • Device: PC, Mac laptop, or Chromebook. On Chromebooks, enable Linux (Beta) or use web IDEs (e.g., Codespaces).

Submission & Review: Students work from their hub, submit GitHub Pull Requests, and discuss blockers in Discord. Sprints move cards Backlog вҶ’ In Progress вҶ’ Done; вҖңClose SprintвҖқ triggers parent notification.
