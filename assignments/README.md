This folder contains shared assignments, homework, and challenges.

## Workflow

1.  **Parents**: Create a new folder for each assignment (e.g., `/week-1-math-quiz`, `/python-calculator-project`). Inside the folder, drop a document detailing the requirements.
2.  **Caleb & Elijah**: When a new assignment is posted:
    *   Read the instructions carefully.
    *   If you need to write code or create documents for the assignment, do so in your personal workspace (`/students/your-name`), *unless* the instructions specify otherwise.
    *   If it's a collaborative project, work within the assignment folder or as directed.
