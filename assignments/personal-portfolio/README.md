# 💻 Personal Portfolio Website

**Grade Level:** Middle School (adjust to your student)
**Subject:** STEM / Computer Science

## 🎯 Objective / What You Will Learn
*   How to structure a multi-page website using HTML.
*   How to link separate HTML pages together using the `<a>` tag.
*   How to apply basic CSS styling (colors, fonts, margins, padding) to make a site look professional.
*   How to organize a digital workspace folder with an `index.html`, style sheet, and images.

---

## 🛠️ Required Resources
*   VS Code (or any preferred text editor)
*   A web browser for testing
*   A newly created folder named `my-portfolio`

---

## 🚀 The Project Details

Instead of just looking at other people's websites, you are going to build your own corner of the internet! You will create a 3-page "Personal Portfolio" that showcases who you are, what you've learned so far, and how people can contact you.

### Step 1: The Folder Structure
Create a new folder called `my-portfolio`. Inside it, create the following files:
*   `index.html` (This will be your Homepage)
*   `projects.html` (Where you show off your 3-Car Collection and Budget Tracker)
*   `contact.html` (A simple page to say hello)
*   `style.css` (The file that will make everything look cool)

### Step 2: The Homepage (`index.html`)
1.  Set up your standard HTML boilerplate (`<html>`, `<head>`, `<body>`).
2.  Link your `style.css` file in the `<head>`.
3.  Add an `<h1>` tag with your name: "Welcome to [Your Name]'s Portfolio".
4.  Write a short paragraph `<p>` introducing yourself and your favorite subjects.
5.  Create a navigation menu using `<ul>` and `<li>` lists. Use `<a>` tags to link to your `projects.html` and `contact.html` pages.

### Step 3: The Projects Page (`projects.html`)
1.  Set up the HTML boilerplate and link your CSS file.
2.  Add a navigation menu at the top so you can get back to the Homepage.
3.  List out at least two projects you've worked on recently (e.g., The Perfect 3-Car Collection, How I Spent My First Million).
4.  Write a 2-sentence description of what you learned during each project.

### Step 4: The Contact Page (`contact.html`)
1.  Set up the HTML boilerplate, link CSS, and add the navigation menu.
2.  Add an `<h2>` that says "Get In Touch".
3.  (Optional Challenge): Create a simple HTML `<form>` with text inputs for a Name and a Message, and a Submit button. (Note: The button won't actually send an email yet, but getting the form to show up on the screen is the goal!)

### Step 5: Styling with CSS (`style.css`)
1.  Change the `background-color` of the `body` to something other than white.
2.  Change the `font-family` to your favorite system font (like Arial, Helvetica, or sans-serif).
3.  Make your `<h1>` tags a different color than your paragraph text.
4.  Add `padding` or `margin` to your text so it doesn't touch the very edge of the browser window.

---

## 📦 Deliverables
*   A folder containing `index.html`, `projects.html`, `contact.html`, and `style.css`.
*   The links between all three pages must work perfectly when clicked in the browser.
