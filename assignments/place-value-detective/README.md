# 🔎 Place Value Detective

**Grade Level:** Elementary (adjust to your student)
**Subject:** Math (Place Value & Rounding)

## 🎯 Objective / What You Will Learn
* Identify the value of digits in multi‑digit numbers.
* Round to the nearest ten and hundred using place value logic.
* Explain your thinking with quick, clear reasoning.

---

## 🛠️ Required Resources
* Pencil and paper (or whiteboard)
* Optional: Highlighters in two colors

---

## 🚀 The Project Details

### Step 1: Place Value Badge
Pick any 5 numbers between 1,000 and 9,999. For each number:
1. Write it in standard form (e.g., 4,527).
2. Underline the hundreds digit and write its value (e.g., 500).
3. Circle the tens digit and write its value (e.g., 20).

### Step 2: Rounding Report
For each number from Step 1, round to:
* The nearest ten
* The nearest hundred
Then, write one sentence for each explaining why (e.g., “Because the tens digit is 7, I round the hundreds up to 500”).

### Step 3: Detective Word Problem
Create one short word problem where someone needs to round a number to help with a real‑life decision (e.g., “About how many miles?”). Provide the rounded answer and explain your choice.

---

## 📦 Deliverables
* A single page (or photo/PDF) showing Steps 1–3.
* Clear rounding sentences that show the rule used.

---

## 📋 [For Parents] Evaluation Checklist
* [ ] Correctly identifies place values for hundreds and tens.
* [ ] Correct rounding to nearest ten and hundred with explanations.
* [ ] Word problem is realistic and the rounding choice is justified.
* [ ] Neat work and complete sentences.
